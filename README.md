# Collection Privée — Système de Gestion des Parfums

Système de gestion de stock pour la vente de parfums **Collection Privée**.

## Fichiers

- `parfums.json` — Base de données des parfums (stock, prix, statut)
- `catalogue.html` — Catalogue HTML pour présentation des parfums
- `update.py` — Script de mise à jour du stock
- `update-visuel.py` — Script de génération des visuels
- `whatsapp-status.jpg` — Image de statut WhatsApp
- `template-reference.jpg` — Image de référence pour les visuels

## Usage

```bash
python3 update.py           # Mettre à jour le stock
python3 update-visuel.py    # Générer les visuels
```
